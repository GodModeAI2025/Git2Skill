# Git2Skill

Git2Skill ist ein Projekt, um aus bestehenden Git-Repositories strukturierte, wiederverwendbare Skills abzuleiten.

## Ziel

- Repositories analysieren und relevante Arbeitsablaeufe identifizieren
- Wissen in klare Skill-Strukturen ueberfuehren
- Skills nachvollziehbar dokumentieren und versionieren

## Projektstatus

Das Repository befindet sich im Aufbau.
